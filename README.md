Цель проекта — создать 2D-игру с прыжками, врагами, сбором монет и другими элементами, характерными для жанра платформеров.
Планы по разработке
В рамках этого проекта я планирую реализовать следующие функции и компоненты:

Перемещение персонажа:
Горизонтальное движение (влево/вправо), прыжки с гравитацией и приземлением на платформы.
Коллизии и физика:
Корректное взаимодействие персонажа с окружением: платформы, стены, смертельные зоны.
Сбор предметов:
Монеты, бонусы и другие коллекционные элементы, увеличивающие счёт.
Враги и препятствия:
Простые противники, которые могут двигаться по заданной траектории или реагировать на игрока.
Уровни и прогрессия:
Несколько уровней с постепенно возрастающей сложностью, переход между ними.
Графика и анимации:
Спрайты для игрока, врагов, фонов и эффектов, а также базовые анимации (например, ходьба, прыжок).
Звук и музыка:
Звуковые эффекты для прыжков, сбора монет, победы и поражения.
Интерфейс пользователя:
Стартовый экран, экран счёта, сообщения об окончании уровня или игры.
